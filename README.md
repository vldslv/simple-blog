# simple-blog
A simple blog, where you can post your articles, create groups, add comments, and subscribe to favorite authors.

Some modules and frameworks that were used for this project:<br>
<ul>
  <li>Bootstrap</li>
  <li>unittest</li>
  <li>tempfile</li>
  <li>shutil</li>
  <li>SimpleUploadedFile</li>
  <li>Paginator</li>
  <li>LocMemCache</li>
  <li>flatpages</li>
  <li>EmailBackend</li>
</ul>
